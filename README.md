# HR-Analytical-dashboard
📊 Project Overview
This Power BI Dashboard provides a high-level executive summary of a multinational corporation's human capital. It is designed to help HR Directors and Executives monitor headcount distribution, diversity metrics, and compensation across various departments and job roles.

🚀 Key Features
Interactive Department Analysis: A Treemap visualization that allows for instant comparison of department sizes (IT, Sales, HR, etc.).

Diversity Tracking: A Donut Chart providing real-time Gender Diversity ratios across the entire organization.

Executive Metrics: KPI Cards highlighting Total Compensation and Total Headcount.

Dynamic Filtering: An interactive Job Role Slicer that allows users to drill down into specific cohorts (e.g., viewing diversity specifically for "Managers" or "Engineers").

🛠️ Tools Used
Power BI Desktop: For data modeling and visualization.

DAX (Data Analysis Expressions): Used to create custom measures like Headcount.

Excel/CSV: Source dataset containing 1,000+ employee records.

💡 Business Insights
Headcount Distribution: Quickly identify which departments are over or under-staffed relative to company goals.

Compensation Parity: Analyze total investment by education level and role to ensure competitive market positioning.

Diversity & Inclusion: Monitor the gender split to meet corporate social responsibility (CSR) targets.

📂 How to View
Download the MNC_Workforce_Dashboard.pbix file from this repository.

Open the file using Power BI Desktop.

Interact with the Job Role Tiles at the bottom to filter the visual data.
